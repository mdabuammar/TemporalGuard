# TemporalGuard

TemporalGuard is a scaffold for a temporal answer verification system.

## Structure

- `skills/`: design notes for each TemporalGuard skill
- `src/temporalguard/`: Python package with pipeline, search, API, data, and reporting modules
- `tests/`: test suite placeholders and smoke tests
- `docs/`: architecture and annotation documentation
- `data/evaluation/`: benchmark and annotation datasets

## Quick start

```bash
python -m app
```
